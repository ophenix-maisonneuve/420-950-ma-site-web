# Épreuve finale

![La Gencive Zen](../assets/images/gencivezen.png)

## Modalités
- L'examen est d'une durée de 3h (12:10 à 15:10)
- Toute documentation est **permise**
- L'accès à internet et à tous vos outils est également **permis**
- La communication entre personnes (étudiants ou autres) est **interdite**
- La caméra web doit demeurer **allumée** en tout temps pendant l'examen
- Les réponses doivent être écrites **dans vos mots**. Un copier-coller évident d'un site web ou d'une réponse d'IA générative sera pénalisé.

## Directives

### 1. Clonez le dépôt de l'exercice

```bash
git clone git@github.com:ophenix-420-930-ma-24636/examen-final.git
```
ou
```bash
git clone https://github.com/ophenix-420-930-ma-24636/examen-final.git
```

### 2. Répondez aux questions de l'énoncé
- L'énoncé est situé sous `docs/examen.md`
- Répondez aux questions directement dans le fichier


## Remise
- La remise s'effectue sur [Léa](https://cmaisonneuve-lea.omnivox.ca/) dans la section *Travaux - Énoncé et remise* (Épreuve finale).
- Vous devez remettre un seul fichier .zip contenant :
   - Le code source de l'application contenant vos améliorations
   - Le fichier `docs/examen.md` contenant vos réponses aux questions
   - Le diagramme UML demandé
      - Ce diagramme peut être fourni en tant qu'image dans un fichier séparé dans le sous-répertoire `docs`
      - Ce diagramme peut aussi être fourni en format Mermaid directement dans le fichier `docs/examen.md`.
