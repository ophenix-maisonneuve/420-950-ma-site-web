# Patrons structurels

Les patrons structurels décrivent comment assembler et organiser des classes ou des objets pour former des structures plus complexes qui demeurent flexibles. Ils s’intéressent principalement aux relations entre les classes.

## Problèmes résolus
Ils facilitent la composition, l’adaptation ou l’extension de structures d’objets sans modifier leur code interne. Ces patrons permettent aussi de réduire les dépendances rigides et d’encourager la réutilisation.
