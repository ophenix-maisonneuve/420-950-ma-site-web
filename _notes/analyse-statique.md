# Analyse statique du code

Dans son sens large, l'**analyse statique** du code inclut toutes les techniques qui permettent d'analyser une application sans l'exécuter, c'est-à-dire sans avoir à interagir avec ses composantes fonctionnelles. Généralement, les formes d'analyse statique s'intéressent plutôt au code source d'une application ou à la façon dont elle inclut et gère ses dépendances.
